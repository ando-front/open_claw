# OpenClaw 基本操作 & 活用ガイド

OpenClawは、単なるチャットAIではなく、あなたのPCやサーバー上でツールを使いこなし、実世界のタスクを実行する「エージェント」です。このガイドではそのポテンシャルを引き出す方法を解説します。

---

## 1. 基本的なインターフェース

OpenClawには3つの主要な操作方法があります。

### A. Terminal (CLI) - 設定と管理
ターミナル上で全体の管理を行います。
*   `openclaw status` : 動作状況の確認。
*   `openclaw doctor` : システムの診断と自動修復。
*   `openclaw dashboard` : Web管理画面を開く（スキルの管理などが可能）。

### B. OpenClaw Chat - ターミナルでの対話
`openclaw chat` を実行すると、ターミナル上でGeminiと直接対話できます。

### C. 外部チャネル (LINE, Telegram等)
一度設定すれば、スマホのLINEから自宅のPCで動くOpenClawを操作できます。

---

## 2. 「スキル (Skills)」でできること

OpenClawの真骨頂は「スキル」によるPC操作です。

| カテゴリ | できることの例 |
| :--- | :--- |
| **ファイル操作** | ローカルのPDFやExcelを読み取る、ファイルの移動、書き換え |
| **ブラウザ操作** | 最新ニュースの検索、Webサイトのスクレイピング、情報の要約 |
| **コマンド実行** | Pythonスクリプトの実行、gitの操作、システム情報の収集 |
| **外部連携** | メール送信、カレンダー予約、Notionへの記録 |

### スキルのインストール例:
```bash
clawhub install web-search  # Web検索スキルを導入
```

## 3. 実践的な活用シーン

### ケース1: 外出先からのPC操作 (LINE経由)
「昨日作成した『企画書.docx』を要約して送って」とLINEで送ると、OpenClawがPC内を検索し、内容を読み取ってLINEへ返信してくれます。

### ケース2: 自動デバッグ・コード解説
ローカルのプロジェクトフォルダで `openclaw chat` を開き、「このリポジトリの構造を教えて」や「エラーコードXXの修正案を考えて」と依頼すると、ファイルを直接読み取って提案してくれます。

### ケース3: 定期的なリサーチ
「毎朝9時に、OpenAIとGoogleの最新のニュースを調べて3行でLINEして」といった自動ワークフロー（近日対応予定のProactive Workflows機能）の構築が可能です。

---

## 4. チャット中に使える便利なコマンド

対話中に以下の「スラッシュコマンド」を入力することで、AIの挙動を制御できます。

*   `/help` : 利用可能なコマンド一覧。
*   `/status` : 現在のAIモデルや利用状況の確認。
*   `/skill <名前>` : 特定のスキルを強制的に実行。
*   `/context` : AIが現在何を「前提知識」として持っているかを確認。
*   `/approve` : AIが重要な操作（ファイルの削除など）をする際の実行許可。

---

## 5. 困ったときは

システムが不安定になった場合は、まず以下のコマンドを試してください。
```bash
openclaw doctor --fix
```
ほとんどの設定ミスやパスの問題は、このコマンドが自動で解決してくれます。
